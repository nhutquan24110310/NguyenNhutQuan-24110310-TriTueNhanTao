# NguyenNhutQuan-24110310-TriTueNhanTao

BTVNCaNhan + AndOrSearch: Gồm 4 nhóm thuật toán đầu
CSP: thuộc nhóm 5
BTVNNhom6: thuộc nhóm 6

Yêu cầu
Python 3.10+ (khuyến nghị)
PyQt5
PyQtChart

1. Cài đặt thư viện
Mở Visual Studio Code.
Mở Terminal:
Terminal → New Terminal
Hoặc nhấn:
Ctrl + `
Sau đó chạy:
pip install PyQt5
Tiếp tục:
pip install PyQtChart
Hoặc cài cùng lúc:
pip install PyQt5 PyQtChart

2. Chạy chương trình
Trong Terminal:
python main.py
Hoặc nếu máy sử dụng lệnh python3:
python3 main.py
Hoặc chạy trực tiếp trong visual studio code
